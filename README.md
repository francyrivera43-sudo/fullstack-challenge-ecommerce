# Fullstack Challenge - E-commerce

This project is a React-based e-commerce application developed as part of a fullstack development challenge.

## Features
- Dynamic product listing using FakeStoreAPI
- Shopping cart with persistence (Zustand + LocalStorage)
- Real-time search and category filtering
- Responsive design with Tailwind CSS
- Atomic Design architecture

## Technologies Used
- React
- Vite
- Zustand
- Tailwind CSS
- Axios
- Lucide React

## Getting Started
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
